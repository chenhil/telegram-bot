# VeryUsefulBot


## Starting
1. Install pipenv
```shell
pip3 install pipenv
```

2. Install package dependencies from pipfile
```shell
pipenv install
```

3. Start the bot
```shell
pipenv run python bot.py
```

## Usage
Add commands below

```
roll - roll a dice
p - get price for a crypto coin
upcoming - get news related to a specific coin
meme - send a dank meme
```
